
1- entrar na pasta do projeto 
2- abrir o terminal como adm e executar o docker compose (docker-compose up -d)
3- abrir o vscode e abrir a pasta do front (Tarefas.Frontend)
4 - abrir um terminal dentro da pasta 
5 - executar o comando para instalação do angular (npm install -g angular-cli)
6 - abrir o visual studio e executar o projeto 
7 - executar o comando dentro do vscode dentro do terminal (ng serve)
